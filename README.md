# Primo Finances

**Primo Finances** es una aplicación web para gestionar y visualizar tu portafolio de inversión de forma clara, intuitiva y visual.

## 🚀 Iniciar el proyecto

```bash
npm install
npm run dev
```

Abre `http://localhost:5173` en tu navegador.

## 🛠️ Tecnologías

- React + Vite
- TailwindCSS
- Recharts
- shadcn/ui
